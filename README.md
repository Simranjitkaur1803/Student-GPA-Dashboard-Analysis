# Student-GPA-Dashboard-Analysis

## Project Overview
This project analyzes retail sales data to understand revenue trends, product performance, and regional sales distribution.  
The goal is to transform raw sales data into clear insights using Excel analysis and Tableau visualization.

## Tools Used
- Excel (data cleaning, pivot tables, exploratory analysis)
- Tableau (interactive dashboard)
- Basic data analysis techniques

## Business Questions
1. Which products generate the highest revenue?
2. Which regions contribute most to overall sales?
3. How do sales change over time?
4. Are there products with high sales but low profit?

## Data Preparation
The dataset was cleaned and prepared using Excel:
- Removed duplicate records
- Standardized product and region fields
- Created calculated fields for revenue and profit analysis

## Dashboard

The Tableau dashboard visualizes:

- Sales by product category
- Sales trends over time
- Regional performance
- Top performing products

![Student GPA Dashboard](Student GPA and Demographics Overview (1).png)

## Key Insights
- A small number of products contribute to a large portion of revenue.
- Certain regions show consistently higher sales performance.
- Sales trends reveal seasonal peaks in demand.
